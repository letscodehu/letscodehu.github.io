# SoundCloud podcast links

The Letscode.hu podcast SoundCloud playlist URL is exposed in i18n (`about.soundcloudUrl` / `about.soundcloudLabel` on the about page; on the Budapest workshop landing it is woven into `trainingB2cAds.instructorBody2Parts` with `instructorLinks.soundcloudUrl` / `soundcloudLabel`). Both entry points open the link in a new tab with `rel="noopener noreferrer"`.
